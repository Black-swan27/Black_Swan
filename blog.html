Introduction to Computing Lessons

1. Computing's Impact on Industries

  • ACM - Association of Computing Machinery.
  • IEEE - Institute of Electrical and Electronics Engineers.
  • IAS - Institute for Advanced Study.
  • Computing - Solving problems or completing tasks.
  • Computer - A programmable device that stores, retrieves, and processes data.

2. Information Processing Cycle

  • Data vs Information:
  • Data - Raw representation of facts and concepts.
  • Information - Organized and meaningful data.
  • Data Processing - Organized and systematic manipulation of information.
  • Good information is relevant, complete, cost-effective, presentable, accurate, and timely.

3. Computer Systems Concepts of Computer Security

  • Systematic interaction between entities.
  • Four components of a computer system: Hardware, Software, Peopleware, and Dataware.
  • Data is raw; information is organized.
  • Topics: Data security, information security, cybercrimes, and cybercriminals.

4. HTML and CSS

  • HTML - HyperText Markup Language.
  • The standard markup language for creating web pages.
  • Describes the structure of web pages using markup.
  • HTML Elements are building blocks of web pages, defined by tags.
  • Tags label content like headings, paragraphs, tables, etc.
  • Browsers do not display HTML tags but use them to render content.
  • CSS - A simple design language used to style HTML elements.
  • CSS allows modifying colors, sizes, backgrounds, layouts, etc.
  • External stylesheets are stored in `.css` files.
  • CSS can be applied using Inline, Internal, or External methods.
  • CSS Selectors are used to target specific HTML elements for styling.
